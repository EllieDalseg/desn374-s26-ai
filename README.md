## About the Project

This is the project website for **CLOSED — 1985**, a fully playable 3D browser-based horror game built entirely using AI — specifically Claude Code. The project was created for AI + Design 374 as an investigation into what AI is truly capable of when it comes to game production, and what it means to co-create something with it.

The site functions as a case study, portfolio piece, and process document — covering the research, development process, final game, and personal reflection on working alongside AI as a creative tool.

## About the Game

**CLOSED — 1985** is a first-person PS1-style horror game set in an abandoned shopping mall on the overnight shift of November 14, 1985. The power is out. The exits are locked. As you explore the corridors of Westfield Mall, you find 7 VHS tapes left behind by former employees — each one revealing more about Harold Finch, the janitor who died in this mall and never left. The final tape gives you the exit code. But the moment you have it, Harold knows.

🎮 **Play it live:** [closed1985.netlify.app](https://closed1985.netlify.app/)
